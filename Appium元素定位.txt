ID定位
Appium中可以使用Find_element_by_id()进行定位
例子：
driver.find_element_by_id(‘android:id/button2’).click()

异常捕捉（类似IF）
from selenium.common.exceptions import NoSuchElementException ←开头声明
例子：
定义一个‘取消’按键的异常捕捉
def check_cancelBtn():
Try:
#定义变量cancelBtn接收‘取消’按键的元素ID
cancelBtn=driver.find_element_by_id(‘android:id/button2’)

#如果当前页面没有‘取消’按键的元素ID，则打印‘no cancelBtn’
except NoSuchElementExceptioon
Print(‘no cancelBtn’)
#如果有则按‘取消’按键
else:
cancelBtn.click()

#调用check_cancelBtn 函数
check_cancelBtn()


name定位
根据name进行定位，对于android来说，就是text属性,用find_element_by_name

例子:
from find_element.capability import driver
driver.find_element_by_name(‘请输入用户名’).send_keys(‘nothing’)
driver.find_element_by_name(‘登录’).click()
说明：由于text稳定性不是很好，所以appium1.5开始废弃了该方法

classname定位
用find_element_by_class_name
Classname定位是根据元素类型来进行定位，但实际情况中很多元素的classname都是相同
很多classname属性值都是相同的，如下例，如果有id就不必使用classname定位

例子:
from find_element.capability import driver
driver.find_element_by_class_name(‘android.widget.EditText’).send_keys(‘nothing’)
driver.find_element_by_class_name(‘android.widget.EditText’).send_keys(‘everything’)

相对定位
相对定位是先找到该元素的对应属性的父元素节点，然后基于父元素进行元素定位

例子:
from find_element.capability import driver
driver.find_element_by_id(‘’).click
定义变量Root_element接收父类id
Root_element=driver.find_element_by_id(‘com.tal.kaoyan:id/activity_register_parentlayout’)
在接收父类的变量中进行元素定位
Root_element.find_element_by_class_name(‘android.widget.ImageView’).click()


xpath定位
Xpath定位是一种路径定位方式，主要是依赖元素绝对路径或者相关属性来定位，但是绝对路径xpath执行效率比较低（特别是元素路径比较深的时候），一般使用比较少。通常使用xpath相对路径和属性定位

1.xpath路径表达式
/    从根节点选取
//   从匹配选择的当前节点选择文档中的节点，而不考虑他们的位置
nodename  选取此节点的所有子节点
.	选取当前节点
..	选取当前节点的父节点
@	选取属性

2.通配符
*	匹配任何元素节点
@*	匹配任何属性节点
node()	匹配任何类型的节点

例子：
#匹配到当前的元素，并他的text值是“请输入用户名”，定位到用户名输入框
driver.find_element_by_xpath('//android.widget.EditText[@text="请输入用户名"]').send_keys('zxw1234')

#匹配当前所有元素，并index=3的，定位到密码输入框
driver.find_element_by_xpath('//*[@class="android.widget.EditText" and @index="3"]').send_keys('zxw123456')

driver.find_element_by_xpath('//android.widget.Button').click()

driver.find_element_by_xpath('//*[@class="android.widget.Button"]').click()




List定位&随机生成
List定位使用find_elements_by_xx获取一组相同的class属性的元素，使用数组下标来区分标记不同元素

例子：
在新用户注册界面点击添加头像按钮后，选择指定的图片保存作为头像
from find_element.capability import driver

driver.find_element_by_id('com.tal.kaoyan:id/login_register_text').click()

driver.find_element_by_id('com.tal.kaoyan:id/activity_register_userheader').click()

#定义一个变量images接收图片
images=driver.find_elements_by_id('com.tal.kaoyan:id/item_image')
#变量images用list保存图片，然后随机点击一张图片
images[10].click()
#保存图片
driver.find_element_by_id('com.tal.kaoyan:id/save').click()
